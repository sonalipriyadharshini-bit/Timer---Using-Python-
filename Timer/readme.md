========================================
   COUNTDOWN TIMER WITH SOUND ALERT
========================================

Project Title:
Countdown Timer with Sound Alert (Line-by-Line Display)

----------------------------------------
Project Description:
----------------------------------------
This project is a simple Countdown Timer 
built using Python. The timer counts down 
from a user-specified number of seconds 
and prints each second on a new line.

When the countdown reaches zero, a sound 
alert is played.

----------------------------------------
Features:
----------------------------------------
- User inputs time in seconds
- Displays countdown in MM:SS format
- Prints each second on a new line
- Sound alert when time is up
- Works in terminal/command prompt
- Beginner-friendly project

----------------------------------------
Technologies Used:
----------------------------------------
- Python 3.x
- time module (built-in)
- winsound module (Windows only)

Note:
No external libraries are required.
All modules used are built into Python.

----------------------------------------
How to Run:
----------------------------------------
1. Open terminal in the project folder.
2. Run the command:

   python countdown_timer.py

3. Enter time in seconds.
4. The countdown will begin.
5. When time finishes, a sound alert plays.
